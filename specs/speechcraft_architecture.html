②
Store
Style vector DB
FAISS · persisted to disk · speech chunks + style feature tags
Content vector DB
FAISS · persisted to disk · topical knowledge from public corpus
Private doc DB
Chroma · in-memory only · per-session · never persisted to disk
Raw doc store
Local filesystem · /tmp for session docs · corpus/ for public
Key decisions: FAISS (not Pinecone) for public corpus — fully local, no cloud. Chroma in-memory (not persisted) for private docs — session cleared after 2 hours. Three separate stores to maintain style/content/privacy separation. Embeddings via OpenAI text-embedding-3-small.
